Веб-сервис для отображения и управления данными о потреблении и ценах на электроэнергию в двух регионах РФ.  
Состоит из FastAPI (бэкенд) и Streamlit (фронтенд). Запускается через Docker Compose.

Сборка и запуск:

1. Убедитесь, что установлен Docker - установить можно тут (https://www.docker.com/products/docker-desktop/)

2. Клонируйте репозиторий и перейдите в корневую папку проекта.

3. Убедитесь, что в папке `backend` присутствует файл `data.csv`.

4. Выполните сборку образов: (bash)
   
  $ docker-compose up --build
